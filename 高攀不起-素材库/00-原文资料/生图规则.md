## 三、提示词写作规范

### 3.1 语言

- **中文**书写
- 专有名词保持中文原样（如：仙侠、暗红灵印、玄冰、寒铁）

### 3.2 必须包含的要素

每条提示词必须包含以下维度：

```
1. 主体描述    — 人物/场景/道具的核心视觉特征
2. 动作/状态   — 人物的表情、姿态、动作瞬间
3. 光影氛围    — 光源方向、色温、冷暖对比
4. 构图/景别   — 特写/中景/全景，镜头角度
5. 风格标识    — 固定尾缀（见下方）
6. 角色锚定    — 苏然面部特征关键词（见下方）
```

### 3.3 固定风格尾缀

每条提示词末尾必须追加：

```
9:16竖屏，古装写实仙侠风，电影级布光，8K，S+品质，极致细节，画面中不得出现任何文字
```

### 3.4 苏然面部特征（人物一致性锚点）

生成苏然图像时，必须包含以下特征描述：

```
年轻中国男子，面色苍白消瘦，细长凤眼眼尾微垂，眉心一枚竖向菱形暗红灵印，黑色长发，白色薄衫
```

### 3.5 禁止项

| 禁止 | 原因 |
|------|------|
| "血""鲜血"等词 | 触发安全过滤，改用"暗色液体""深红液体""暗红" |
| 字幕/文字 | 画面中不得出现任何文字 |
| 动漫/插画/3D渲染/CG/数字艺术 | 保持写实真人风格 |
| 凭空添加原文没有的道具/人物 | 忠于原文 |

## 六、生图流程

```
1. 确认需求 → 明确要生成什么（角色/场景/首帧/尾帧/参考图）
2. 查原文   → 从小说原文中提取该镜头对应的视觉描写
3. 查素材库 → 确认所需角色/场景 CDN 链接已就绪
4. 写提示词 → 按第三章规范编写中文提示词
5. 提交API  → POST /v1/media/generate，传入提示词 + 参考图 CDN
6. 轮询结果 → GET /v1/media/status?task_id=xxx
7. 下载归档 → 保存到对应目录（04-竖屏首尾帧 或 06-竖屏视频）
```
