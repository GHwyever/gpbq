# 番茄小说 → 短剧改编工作流工具

## 功能概述

本工具集用于帮助已获得抖音短剧中心IP改编授权的制作方，高效地将番茄小说内容转化为短剧改编方案。

**核心功能：**
- 小说内容结构化解析（章节、字数统计）
- 角色自动识别与分析
- 场景提取与情绪识别
- 关键剧情节点标注
- 短剧分集大纲自动生成
- 分镜脚本模板生成
- 改编总结报告导出

## 适用场景

✅ 已获得番茄小说IP改编授权的短剧制作团队
✅ 需要将小说内容系统化分析后进行改编的创作者
✅ 批量处理多部IP的改编前期策划

## 安装

```bash
pip install -r requirements.txt
```

依赖：
- `jieba` - 中文分词与词性标注（用于角色识别）

## 使用方法

### 基本用法

```bash
python main.py 小说文件.txt --title "小说标题" --author "作者名"
```

### 示例

```bash
python main.py my_novel.txt --title "绝世神皇" --author "张三"
```

### 输出文件

运行后会在 `output/` 目录生成：

| 文件 | 说明 |
|------|------|
| `novel_structure.json` | 小说结构化数据（章节、内容预览） |
| `adaptation_report.json` | 完整改编分析报告 |
| `shots/episode_XX_shots.json` | 分集分镜脚本 |
| `改编总结.md` | 改编方案总结文档 |

## 工作流程

```
小说TXT文件
    ↓
[步骤1] 内容解析 → 章节结构、字数统计
    ↓
[步骤2] 智能分析 → 角色、场景、剧情节点
    ↓
[步骤3] 改编规划 → 30集分集大纲
    ↓
[步骤4] 分镜生成 → 每集详细镜头脚本
    ↓
输出完整改编方案
```

## 模块说明

### novel_processor.py

小说内容处理器：
- `NovelProcessor` - 解析TXT格式小说
- 自动识别章节标题（支持"第X章"、"第X回"等格式）
- 内容清洗（去除广告、多余空行）
- 导出JSON结构化数据

### drama_adapter.py

短剧改编辅助工具：
- `DramaAdapter` - 核心改编分析引擎
- `analyze_characters()` - 基于jieba词性标注识别人名
- `extract_scenes()` - 识别场景地点与情绪
- `identify_plot_points()` - 标注关键剧情节点
- `generate_episode_outline()` - 生成30集分集大纲
- `generate_shots_for_episode()` - 生成详细分镜

## 合规声明

⚠️ **本工具仅用于处理合法获取的小说内容**

使用本工具前，请确保：
1. 您已获得番茄小说官方IP改编授权
2. 内容获取方式符合抖音短剧版权中心合作规范
3. 改编作品将按合同约定在指定平台发布

未经授权抓取或传播小说内容可能违反《著作权法》及相关法规。

## 官方合作渠道

如需申请IP改编授权，请通过官方渠道：
- 抖音集团短剧版权中心：https://novelcp.toutiao.com
- 漫剧IP改编库：版权中心平台后台在线申请
- 合作模式：模式二（番茄IP改编合作）或模式三（特定授权）

## 技术说明

- 角色识别基于jieba的`nr`（人名）词性标注，可能存在误识别
- 场景识别基于关键词匹配，建议根据实际内容调整`LOCATION_KEYWORDS`
- 分镜生成为模板化输出，实际制作需结合创意调整
